## Prepare Demo Data

- demo learner account
- optional demo admin account for live interviews
- sample lesson
- sample questions
- no real user data
- no private exports or logs

## Demo Flow

1. Explain the problem.
2. Show dashboard and onboarding.
3. Open a lesson.
4. Explain answering and scheduling.
5. Show learning settings.
6. Show learning statistics.
7. Show admin question editing.
8. Mention tests and quality checks.

## Safety Notes

Do not publish:

- real credentials
- production database details
- private source code
- real user data
- full admin access
