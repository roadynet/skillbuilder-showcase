# Demo Checklist

## Prepare Screenshots

Recommended screenshots:

1. Dashboard
2. Lesson overview or detail page
3. Learning statistics
4. Learning settings
5. Admin question editor
6. User management
7. Account / GDPR area with demo data only

Place screenshots in `screenshots/`.

## Prepare Demo Data

- demo learner account
- optional demo admin account for live interviews
- sample lesson
- sample questions
- no real user data
- no private exports or logs

## Demo Flow

1. Explain the problem.
2. Show dashboard and onboarding.
3. Open a lesson.
4. Explain answering and scheduling.
5. Show learning settings.
6. Show learning statistics.
7. Show admin question editing.
8. Mention tests and quality checks.

## Safety Notes

Do not publish:

- real credentials
- production database details
- private source code
- real user data
- full admin access
